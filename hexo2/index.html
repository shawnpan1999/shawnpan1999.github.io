Hexo博客搭建：(二)Next主题配置

发表于 2020-01-30 | 分类于 Hexo博客搭建 |

了解 Next 主题配置文件 blog\themes\next\_config.yml 中客制化的功能，个性化你的博客。

博主使用的 Next 版本为 V5.1.4 ，不同版本的配置文件可能有少许差异。

推荐一个可以get到免费图标的网址：flaticon

网站页脚

这部分修改的是网站最下方的版权和 powered by 的内容啦。

菜单栏

食用方式：Key: /link|| icon 注意这里的 link 之后不要跟上斜杠和空格，直接跟 || ，否则可能出现定位到 /link/%20/ 之类的奇怪错误。

key 为某个菜单项的标识，这个标识和 zh-Hans.yml 里的标识相对应。

link 为链接地址。

icon 为图标名称，同样可以在 图标库 中查询。

增加新菜单：把对应菜单去注释（这里没有就自己输入一行），使用命令 hexo new page "xxx" 即可增加一个新的页面，最后记得要在 blog\themes\next\languages\zh-Hans.yml 里汉化它。
