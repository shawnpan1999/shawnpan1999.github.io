可能遇到的坑以及解决方案

各种 command not found / 不是内部或外部命令

检查一下该软件是否已经加入系统变量的 path 中。

解决方法：

此电脑右键 -> 高级系统设置 -> 环境变量 -> 系统变量 Path。把它添加进去即可。

permission to xxx denied to yyy

我是在执行 hexo d 的时候出现这个报错。原因是我之前使用另一个 github 账号A登录过，系统已经将 github.com 的用户凭据保存下来了。此时默认使用A账户进行操作，自然就被卡在权限外了。

解决方法：

控制面板 -> 用户账户 -> 管理 Windows 凭据 -> 普通凭据

找到 github.com ，更新一下用户名和密码即可。
