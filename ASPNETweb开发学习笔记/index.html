ASP.NET web开发学习笔记

发表于 2020-03-01 | 分类于 学习笔记 |

以前上课时记的笔记，ASP.NET 胜在简单方便，对于 web 初学者而言，可以很容易地捏出像模像样的页面，获得学习反馈 (成就感) 是很容易的。做这个 web 项目使我掌握了 html 、css 、JavaScript 这些前端技术以及 c# 后台、数据库等知识。

WebForm开发模式

WebForm与MVC

WebForm 采用事件驱动模式来控制应用的请求，提供了VS的可视化编辑界面(拖一个 Button 到页面中，双击 Button 就可以编辑点击事件)；同时有大量的服务器控件支持，很多东西 WebForm 已经帮我们封装好了，直接就可以拿来使用。这使得应用的开发过程变得相当快捷便利。

相比之下，MVC通过Model，View和Controller分解了复杂的架构，同时很好地符合了“高内聚，低耦合”的原则。更利于大型项目的团队的分工合作。

Session 记录当前用户身份

Session 是在服务器端的内存中存储信息。在我们实现页面保护时，可以在用户登录后建立 Session[用户名] ，在页面加载时判断这个 Session 的状况从而决定是否展示页面。
