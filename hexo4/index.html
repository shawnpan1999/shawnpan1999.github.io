（四）绑定域名

发表于 2020-02-08 | 分类于 Hexo博客搭建 |

常见的域名商有国内的阿里云、腾讯云，国外的 GoDaddy 等等……因为博主使用的是 GitHub Pages + 静态页面，没有使用大陆服务器，因此可以不用登记备案，买完域名直接解析到博客地址即可。

添加解析

登录到腾讯云的控制台，点击域名解析列表 -> 添加记录：

主机记录：可以理解为域名前缀。www 可以解析三达不溜开头的域名；@ 代表无前缀

记录类型：填 CNAME ，因为待会要将它指向我们的 .github.io 域名

记录值：就是博客的地址，一般是 username.github.io

建议建立两条解析记录，分别记录 www 和 @ 这两种域名前缀

新建 CNAME 文件

在目录 blog/source 下新建无后缀文件，命名为 CNAME 。

文本编辑器打开，输入域名 www.psw.life ，保存。

更改好后不忘 hexo g 、 hexo d 二连，提交更改。

设置 GitHub Pages 域名

前往博客项目的 GitHub 地址 -> 设置 -> GitHub Pages

把 Custom domain 这一项改成域名就好了喔~

GitHub Pages 每次更新后大概需要几分钟到十几分钟不等的时间来部署，等到出现如图中的绿色提示 Your site is published at xxx 即可使用新域名访问博客了~
